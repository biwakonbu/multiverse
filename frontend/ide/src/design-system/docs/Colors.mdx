# Color Palette

Multiverse IDE uses the **Nord Deep** theme, extending the Nord palette with deeper background tones for a rich, immersive environment.

## Semantic Colors (Recommended)

Always use semantic colors in your components. These map to the primitive colors but define the _purpose_ of the color.
