# Spacing & Layout

Spacing tokens govern the padding, margin, and gaps between elements.
Multiverse uses a 4px-based scale for UI components and specific grid units for the node graph.

## Grid Layout System

Specialized units for the WBS and Graph layout.

| Token            | Value                         | Description                     |
| :--------------- | :---------------------------- | :------------------------------ |
| `cell-width`     | `{grid.cellWidth}px`          | Standard width of a task node   |
| `cell-height`    | `{grid.cellHeight}px`         | Standard height of a task node  |
| `cell-gap`       | `{grid.gap}px`                | Gap between grid cells          |
| `toolbar-height` | `{layout.toolbarHeight}px`    | Fixed height of the top toolbar |
| `sidebar-width`  | `{layout.detailPanelWidth}px` | Width of the WBS panel          |

## Z-Index Structure

| Layer     | Value  | Purpose                   |
| :-------- | :----- | :------------------------ |
| `base`    | `0`    | Default content           |
| `node`    | `10`   | Grid nodes                |
| `panel`   | `100`  | Floating panels, toolbars |
| `modal`   | `900`  | Modals, overlays          |
| `tooltip` | `1000` | Tooltips, popovers        |
| `cursor`  | `9999` | Custom cursor             |
