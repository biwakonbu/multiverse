# Typography

Multiverse IDE uses a focused set of typefaces to distinguish between functional UI (Sans), Branding (Orbitron), and Code (Mono).

## Font Families

<Source
  code={`

`}

/>

## Font Weights

- **Normal (400)**: Default text
- **Medium (500)**: Labels, emphasis
- **SemiBold (600)**: Subheadings, important buttons
- **Bold (700)**: Headings, Milestones

## Line Heights

- **Tight (1.2)**: Headings, single-line labels
- **Normal (1.5)**: Body text
- **Relaxed (1.75)**: Long-form reading
